Все значения в переменных можно изменять, перезаписывать. const - переопределять нельзя. 

Типы данных:

int = целые числа

uint = только положительные целые числа

byte = только целые числа, которые не привышают 255(байт
позволяет нам экономить потребление операционной памяти в 
отличии от int. При использования int задействуется больше 
памяти, что замедляет программу.)

short = это второй по величине тип данных, который может содержать числа не привышающие 32767.(он уже использует вдва раза больше оперативной памяти, но все же меньше чем int)

long = самое длинное число

float = число с точкой. нецелочисленные числа, но нам в конце каждой цифры нужно указывать букву f. 
float num = 2.33435f

double = тоже число с плавающей точкой, но большее по значению. также в конце указываем букву d. 
double num 343434.334343433226667d


string = "Обязательно должны быть двойные ковычки"

char = используется для вывода только одного символа, в этом случае мы используем одинарные ковычки ''. 

bool = true of false.
